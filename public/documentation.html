Movie API Documentation

Endpoints

Request URL Response
GET /movies Returns a list of ALL movies
GET /movies/:title Returns data (description, genre, director, image URL, whether it's featured or not) about a single movie by title
GET /genres/:name Returns data (description) about a genre by name
GET /directors/:name Returns data (bio, birth year, death year) about a director by name
POST /users Allows new users to register
PUT /users/:username Allows users to update their user info (username)
POST /users/:username/movies/:movieID Allows users to add a movie to their list of favorites (showing only a text that a movie has been added)
DELETE /users/:username/movies/:movieID Allows users to remove a movie from their list of favorites (showing only a text that a movie has been removed)
DELETE /users/:username Allows existing users to deregister (showing only a text that a user email has been removed)

Endpoint Details

GET /movies

Returns a list of ALL movies.

URL: /movies

Method: GET

Query Parameters: None

Response Format: JSON array of movie objects

GET /movies/:title

Returns data (description, genre, director, image URL, whether it's featured or not) about a single movie by title.

URL: /movies/:title

Method: GET

Query Parameters: title (string)

Response Format: JSON object of movie data

GET /genres/:name

Returns data (description) about a genre by name.

URL: /genres/:name

Method: GET

Query Parameters: name (string)

Response Format: JSON object with genre description

GET /directors/:name

Returns data (bio, birth year, death year) about a director by name.

URL: /directors/:name

Method: GET

Query Parameters: name (string)

Response Format: JSON object with director data

POST /users

Allows new users to register.

URL: /users

Method: POST

Body Format: JSON object with user data

Response Format: Text confirming registration

PUT /users/:username

Allows users to update their user info (username).

URL: /users/:username

Method: PUT

Query Parameters: username (string)

Body Format: JSON object with updated user data

Response Format: Text confirming update

POST /users/:username/movies/:movieID

Allows users to add a movie to their list of favorites.

URL: /users/:username/movies/:movieID

Method: POST

Query Parameters: username (string), movieID (string)

Response Format: Text confirming addition

DELETE /users/:username/movies/:movieID

Allows users to remove a movie from their list of favorites.

URL: /users/:username/movies/:movieID

Method: DELETE

Query Parameters: username (string), movieID (string)

Response Format: Text confirming removal

DELETE /users/:username

Allows existing users to deregister.

URL: /users/:username

Method: DELETE

Query Parameters: username (string)

Response Format: Text confirming de-registration
